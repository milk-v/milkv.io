# Jupiter2

## Jupiter2 简介

Jupiter2 是面向边缘 AI、机器人与工业控制场景的高性能 RISC-V 单板计算机，搭载 SpacemiT K3 八核处理器，提供 60TOPS 通用 AI 算力，并支持 LPDDR5 高速内存与 UFS 本地存储。主板采用 Pico-ITX plus 紧凑形态，集成万兆 SFP+ 光口、千兆以太网、WiFi 6/BT5.2、双 M.2 高速扩展、DP/eDP 显示输出及 FPC 实时扩展接口，可兼顾 AI 推理、高速数据传输、运动控制和多系统部署需求，适合行业客户快速完成智能终端、边缘计算设备和工业系统集成。
