## 准备工作

### 必要

1. Titan 主板
2. 内存：DDR4 内存两条，两个内存插槽都需要安装内存条，并且确保所使用的两条内存规格一致，具体规格请参考该链接：[内存支持列表](https://milkv.io/zh/docs/titan/getting-started/memory)
3. 系统启动介质：NVME SSD（用于安装系统与启动）
4. 电源：主板支持 ATX（24针）和 DC（5525） 两种供电方式。
   Titan 主板功耗取决于所连接的外设功率和数量，需根据具体情况选择合适的供电方式。以下是在满足系统可正常运行状态下，空载和 CPU 压力测试时的功耗，仅供参考。如果您还连接了其他外设，比如外接显卡，还需要考虑显卡本身的功耗

   - DDR4 64GB(32GB*2) + M.2 NVMe 固态硬盘(128G)，空载功耗约14W(12V/1.2A)
   - DDR4 64GB(32GB*2) + M.2 NVMe 固态硬盘(128G)，满载功耗约30W(12V/2.5A)

:::tip
Titan 主板本身没有 HDMI 等显示接口，在其做为服务器等无显示输出需求的场景下，可无需连接独立显卡和显示器。设备支持通过 USB Debug 串口或 SSH 网络协议进行登录与管理。
:::

### 可选

1. 显卡：Titan 板载一个 PCIe Gen4 x16 的插槽，在使用桌面系统时，可通过独立显卡连接显示器使用，我们验证过的显卡如下，仅供参考：
   - RX 9070 XT
   - RX 580
   - RX 550 （昂达RX550 典范 4GD5 V5）
   - R5 230 （R5 230 2G DDR3）
2. 显示设备：在使用独立显卡时，根据显卡接口选择合适的 HDMI/DP 数据线和显示器
3. 输入设备：键盘和鼠标（用于图形界面操作）
4. 调试工具：USB Type-C 数据线（用于串口登录或调试）
5. 机箱：Titan 主板尺寸符合 Mini-ITX 规范，您可以选择兼容 Mini-ITX 主板的机箱
