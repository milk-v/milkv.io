# 内存支持列表

:::tip
Titan 支持 DDR4 UDIMM。两个内存插槽都需要安装内存条，并且确保所使用的两条内存规格一致。
:::

| 品牌 | 型号 | 电压 | 内存大小 |
|:-------------|:-----:|:-------:|:---------|
| 英睿达(Crucial) | DDR4-3200 UDIMM CL22 | 1.2V | 8GB*2 |
| 英睿达(Crucial) | DDR4-3200 UDIMM CL22 | 1.2V | 32GB*2 |
| 三星(SAMSUNG) | 2Rx8 PC4-3200AA-UB1-11 | 1.2V | 16GB*2 |
| 三星(SAMSUNG) | 2Rx8 PC4-3200AA-UB2-11 | 1.2V | 32GB*2 |
| 海力士(SKhynix) | 1Rx8 PC4-3200AA-UA3-12 | 1.2V | 16GB*2 |
| 佰维(BIWIN) | 1R×8 PC4-3200AA-CL22-UA3 | 1.2V | 16GB*2 |
| 雷克沙(Lexar) | 2Rx8 3200MT/s CL22 | 1.2V | 32GB*2 |
| 威刚万紫千红(ADATA) | DDR4 3200(22) 32Gx16 U-DIMM | 1.2V | 32GB*2 |
