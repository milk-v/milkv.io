# BMC 概述

## 简介

BMC（Baseboard Management Controller）基板管理控制器是服务器中专用于硬件监控与管理的独立单元，具备远程管理、持续监控和故障排除等功能。作为一个独立的子系统，BMC 拥有自己的固件，其运行不依赖于主板上的 CPU、EC 等其他单元，但可与这些单元交互，实现对服务器的全面监控与管理。

Titan BMC 是专为 Milk-V Titan 系统设计的 Web 管理平台。它提供了直观的界面来管理你的 Titan 主机，主要功能包括：

- **远程控制台访问**：通过 UART 或 TTY 连接到主机终端，无需物理接触
- **电源管理**：远程控制主机电源状态（启动、关闭、重启）
- **系统监控**：查看实时状态和资源信息
- **Agent 部署**：轻松安装和管理主机代理以增强功能
- **Web 界面**：从任何设备的浏览器管理你的 Titan 系统
