# Jupiter2

## Jupiter2 Introduction

Jupiter2 is a high-performance RISC-V single-board computer designed for edge AI, robotics, and industrial control applications. Powered by the SpacemiT K3 octa-core processor, it delivers 60TOPS of general-purpose AI computing power and supports high-speed LPDDR5 memory and onboard UFS storage. With a compact 2.5-inch Pico-ITX plus form factor, Jupiter2 integrates 10GbE SFP+ optical networking, Gigabit Ethernet, Wi-Fi 6/BT5.2, dual M.2 high-speed expansion slots, DP/eDP display outputs, and FPC real-time expansion interfaces. It supports AI inference, high-speed data transmission, motion control, and multi-system deployment, enabling industry customers to quickly build intelligent terminals, edge computing devices, and industrial system solutions.
